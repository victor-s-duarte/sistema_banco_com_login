# Sistema bancário em Python

# Descrição
    Sistema simples de conta bancária feito somente com Python, a fim de desenvolver e manter afiados meus conhecimentos em Python e lógica de programação.
    
# Funcionalidades
    Login
    Depósito
    Saque
    Transferencia
    Histórico de transações

# Como executar
    Baixe todos os arquivos. Como todos na mesma pasta, abra o arquivo main.py com sua IDE de preferência, e digite no terminal "python main.py"
